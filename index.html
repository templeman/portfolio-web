Madame Olivia

2010 - Present

Madame Olivia is a step away from your run-of-the-mill brochure site. I was approached by Madame Olivia herself to design and build a sophisticated fortune-telling website that would stand out from the crowd. I had numerous consultations with the client before coming up with the strategy and design that is currently employed on the site. The backend is built on Codeigniter and uses PHP to dynamically splice reading components into cohesive, personlized fortunes. The system includes an automated subscription mechanism that alerts visitors by email when their new reading is ready for them to view. In addition to the blog-style advice column, I created a custom admin interface designed to be simple enough for the client to easily edit and submit new readings to the database. This project has followed an agile development timeline, and although it is live I continue to make improvements based on observations of our current user base.

Instruments: PHP + Codeigniter, Sequel Pro, Compass + SASS, HTML, Photoshop, Wordpress
